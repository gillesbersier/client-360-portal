# Client 360 Portal

A unified **Radial Hub-and-Spoke** dashboard for visualizing customer interactions.

## Project Overview
This project implements a portal where the customer entity acts as the "Sun" in a solar chart, with various business domains (Leads, Orders, FAQs, etc.) orbiting as satellite modules.

## Tech Stack
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **Icons:** Lucide React

## Getting Started

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/gillesbersier/client-360-portal.git
    cd client-360-portal
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    ```

3.  **Run the development server:**
    ```sh
    npm run dev
    ```

## Features
- **Radial Layout:** Dynamic positioning of modules around a central hub.
- **Interactive Hub:** Click the central avatar to upload/change the customer image.
- **Responsive Design:** Adapts to different screen sizes.
- **Brand Theming:** Custom color palette (Deep Purples & Pinks).
